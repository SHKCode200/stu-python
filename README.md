# Stu-Python

##  Description
Ce projet est une application simple en Python permettant de gérer les notes des étudiants.
Il a été réalisé dans un cadre d’apprentissage afin de renforcer la compréhension des bases de la programmation en Python, notamment la manipulation des listes, des dictionnaires, des fonctions et la gestion des erreurs.

Le programme permet à l’utilisateur d’entrer les informations des étudiants, de calculer leurs moyennes et d’afficher les résultats avec des mentions.



##  Objectifs du projet
- Comprendre la structure d’un programme Python
- Manipuler les listes et les dictionnaires
- Utiliser les fonctions pour organiser le code
- Gérer les erreurs de saisie avec `try / except`
- Appliquer des conditions (`if / elif / else`)
- Améliorer la logique algorithmique



##   Fonctionnalités
- Ajouter plusieurs étudiants
- Saisir les notes pour chaque matière (entre 0 et 20)
- Vérifier la validité des notes saisies
- Calculer la moyenne de chaque étudiant
- Afficher une mention selon la moyenne :
  - Très Bien
  - Bien
  - Passable
  - Insuffisant
- Afficher la meilleure et la plus faible moyenne



##  Technologies utilisées
- Langage : **Python 3**
- Concepts :
  - Listes
  - Dictionnaires
  - Fonctions
  - Boucles
  - Conditions
  - Gestion des exceptions
